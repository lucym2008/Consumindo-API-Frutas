API desenvolvida para consumo de dados de frutas utilizando uma API pública externa (Fruityvice). O projeto tem como objetivo demonstrar a utilização do Spring Boot com WebClient para integração com serviços externos.

---

## Tecnologias

- Java 17+
- Spring Boot
- Spring WebFlux (WebClient)
- Maven

---

## Práticas adotadas

- Arquitetura em camadas (Controller e Service)
- Consumo de API externa com WebClient
- Injeção de Dependências
- Código simples e objetivo para aprendizado
